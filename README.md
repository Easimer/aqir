# Aqir
## MMORPG Memory Editing Framework
Supported game versions:
* 5.4.8 (18414) x64
### Build
```bash
git clone https://github.com/Easimer/aqir
cd aqir
make -j4
```
### Install
```bash
cp *.so *.sh <game root directory>
```
### Usage
```bash
cd <game root directory>
./run.sh
```
```bash
telnet localhost 1666
```
